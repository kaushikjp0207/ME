# Happy Birthday Kaushikbhai

A cinematic, static birthday experience built with React, Vite, TailwindCSS, Framer Motion, GSAP, Lenis, and Canvas Confetti.

## Install

npm install

## Development

npm run dev

## Build

npm run build

## GitHub Pages deployment

1. Update the repository name in `vite.config.js` if needed.
2. Install dependencies:
   `npm install`
3. Build the site:
   `npm run build`
4. Deploy to GitHub Pages:
   `npm run deploy`
5. In GitHub repository settings, set the Pages source to the `gh-pages` branch.
6. The project also includes a GitHub Actions workflow in `.github/workflows/deploy.yml` for automatic deployment.

This project is designed to be fully static and GitHub Pages friendly.
